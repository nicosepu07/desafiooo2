# 📊 Análisis de Churn — Telecom X

## 📌 Descripción del Proyecto

Este proyecto tiene como objetivo analizar la evasión de clientes (Churn) en Telecom X utilizando técnicas de análisis exploratorio de datos (EDA).

Se busca identificar los factores que influyen en la cancelación del servicio para generar recomendaciones estratégicas orientadas a mejorar la retención de clientes.

---

## 🎯 Objetivo del Análisis

- Calcular la tasa de churn.
- Identificar variables asociadas a la evasión.
- Detectar patrones de comportamiento.
- Generar insights accionables.

---

## 🛠 Tecnologías Utilizadas

- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab
